# Machine Learning Project: Customer Retail Analytics

A comprehensive Machine Learning project that explores, preprocesses, and models a customer retail dataset to predict patterns and compare algorithm performances.

## 📌 Project Objective
The objective of this project is to build and compare multiple Machine Learning models using a customer dataset. This workflow covers preprocessing, visualization, model training, prediction, evaluation, and algorithmic comparison.

---

## 📊 Dataset Features Used
The model leverages key customer transactional metrics and demographic dimensions to extract patterns:
* **Quantity:** The volume of items per transaction.
* **UnitPrice:** The price per unit of the item.
* **Country:** The geographic location of the customer.

---

## ⚙️ Machine Learning Models Implemented
To evaluate and contrast performance across different inductive biases, the project implements:
* Logistic Regression
* Decision Tree Classifier
* K-Nearest Neighbors (KNN)

---

## 🚀 Project Workflow
1. **Load Dataset:** Load the customer dataset using Pandas.
2. **Data Cleaning:** Handle missing values within the dataset.
3. **Feature Engineering:** Encode categorical columns using `LabelEncoder`.
4. **Exploratory Data Analysis (EDA):** Visualize customer data using Matplotlib.
5. **Data Splitting:** Split data into training and testing sets.
6. **Model Training (Logistic Regression):** Train the Logistic Regression classifier.
7. **Model Training (Decision Tree):** Train the Decision Tree classifier.
8. **Model Training (KNN):** Train the K-Nearest Neighbors classifier.
9. **Model Evaluation:** Evaluate models using Accuracy Score and a Confusion Matrix.
10. **Model Comparison:** Compare model performances visually using graphs.

---

## 📈 Evaluation & Visualization
### Evaluation Metrics
* **Accuracy Score:** To quantify overall correctness.
* **Confusion Matrix:** To diagnose misclassifications across categories.

### Visualizations Included
* Customer Distribution Graph
* Model Accuracy Comparison Graph
